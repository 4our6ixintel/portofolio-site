How to Host a Website on GitHub Pages

Want to host your website for free? GitHub Pages lets you do that easily. Here’s how:

  1. Create a GitHub repo and upload your website files (index.html, CSS, etc).
  2. Go to Settings > Pages.
  3. Select the main branch and click save.
  4. Your website will be live at: https://yourusername.github.io/reponame

That’s it! It’s perfect for portfolios, blogs, or small projects.
